

# Plan: Log Out + حوار متبادل مع منصور + تبديل الشخصيات + Reset Password

## فهمت إيه

1. **Log Out**: زرار تسجيل خروج داخل اللعبة
2. **حوار متبادل**: المحادثة مع منصور لازم تكون تبادلية — اللاعب (أحمد/سارة) يرد كمان مش بس منصور يتكلم
3. **تبديل الشخصيات**:
   - **منصور** = يستخدم صورة `ahmed.png` (اللي لابس بدلة وبدقن) بدل `analyst.png`
   - **المحلل (detective)** = يفضل `analyst.png` (اللي لابس قميص كحلي)
   - **أحمد** (الشخصية الموجودة في اللعبة) = محتاج صورة مختلفة عن منصور
4. **Forgot/Reset Password**: إضافة نسيت كلمة المرور + صفحة reset password
5. **إيميل حقيقي**: حاليا auto-confirm مفعل (مش لازم إيميل حقيقي). لو عايز إيميل حقيقي لـ reset password، محتاج نفعل custom email domain — ده ممكن نعمله بعدين بسهولة.

## سؤال الإيميل الحقيقي

حاليا التسجيل بيشتغل **بدون تأكيد إيميل** (auto-confirm). لو عايز forgot password يبعت إيميل حقيقي، محتاج نربط domain إيميل — ده خطوة إضافية سهلة نقدر نعملها بعد كده. دلوقتي نقدر نبني الـ UI بتاع forgot/reset password وهيشتغل لما نفعل الإيميل.

## خطة التنفيذ

### 1. زرار Log Out
- إضافة زرار تسجيل خروج في `Index.tsx` (أيقونة في أعلى الشاشة)

- **`EnhancedDialogue.tsx`**: تغيير ألوان منصور لتكون مختلفة عن أحمد

### 4. Forgot Password + Reset Password
- إضافة رابط "نسيت كلمة المرور؟" في صفحة Auth
- صفحة `/forgot-password`: إدخال الإيميل → `resetPasswordForEmail()`
- صفحة `/reset-password`: تغيير كلمة المرور → `updateUser({ password })`
- Route جديد في `App.tsx`
